/tg/ Station 13 - Modules - Types — Var Details - Proc Details

/datum/element

A holder for simple behaviour that can be attached to many different types

Only one element of each type is instanced during game init. Otherwise acts basically like a lightweight component.

Vars

element_flagsOption flags for element behaviour
id_arg_indexThe index of the first attach argument to consider for duplicate elements Is only used when flags contains ELEMENT_BESPOKE This is infinity so you must explicitly set this
